# 自动求职助手使用说明

## 简介

这是一个自动化求职工具，可以帮助您在LinkedIn等平台上搜索和申请工作。该工具使用AI技术来分析您的简历，寻找匹配的工作机会，并自动填写申请表格。工具会持续运行直到完成100个工作申请。

## 使用前准备

1. **准备您的简历**：
   - 确保您有一份最新的PDF格式简历
   - 将简历命名为`cv_04_24.pdf`并放在与程序相同的文件夹中
   - 如果您想使用不同名称的简历，请修改`config.yaml`文件中的路径

2. **准备头像图片**：
   - 准备一张名为`zhangyixin-touxiang.png`的头像图片
   - 将图片放在与程序相同的文件夹中

3. **安装Chrome浏览器**：
   - 程序需要使用Chrome浏览器
   - 确保您已安装最新版本的Chrome浏览器

## 运行方式选择

您可以选择以下三种方式之一来运行求职助手：

### 方式一：使用独立可执行文件（推荐普通用户）

1. 下载打包好的可执行文件（从发布页面或管理员提供的链接）
2. 解压缩文件到任意文件夹
3. 将您的简历和头像放入同一文件夹
4. 双击运行`求职助手.exe`（Windows）或`求职助手`（Mac）

### 方式二：使用安装脚本（适合有基本电脑知识的用户）

#### Windows用户：
1. 确保已安装Python 3.11或更高版本
2. 双击运行`setup.bat`
3. 按照提示操作

#### Mac/Linux用户：
1. 确保已安装Python 3.11或更高版本
2. 打开终端，进入程序所在文件夹
3. 运行命令：`chmod +x setup.sh && ./setup.sh`
4. 按照提示操作

### 方式三：使用持续运行脚本（适合高级用户）

1. 确保已安装Python 3.11或更高版本和所有依赖项
2. 运行命令：`python continuous_job_search.py`
3. 程序会自动重启直到完成100个申请

## 配置文件说明

程序使用`config.yaml`文件进行配置，您可以修改以下设置：

```yaml
# 工作搜索设置
job_search:
  # 搜索关键词（可添加多个）
  keywords: ["AI agent", "软件工程师", "数据科学家"]
  # 搜索地点（可添加多个）
  locations: ["上海", "北京", "远程"]
  # 最大搜索工作数量
  max_jobs: 100

# 简历设置
resume:
  # 简历文件路径（请确保此文件存在）
  path: "./cv_04_24.pdf"

# API凭证（如果您有API密钥，请填写）
credentials:
  # OpenAI API密钥（可选）
  openai_api_key: ""  
  # Google API密钥（可选）
  google_api_key: ""  
```

## 运行过程

1. 程序启动后会自动打开Chrome浏览器
2. 程序会自动访问LinkedIn等招聘网站
3. 程序会搜索符合您简历的工作机会
4. 程序会自动填写申请表格并提交申请
5. 程序会记录已申请的工作，避免重复申请
6. 程序会持续运行直到完成100个申请或遇到无法解决的错误

## 查看申请结果

程序会生成以下文件来记录申请结果：

1. `job_applications.csv` - 包含所有已申请工作的详细信息
2. `application_stats.json` - 包含申请统计信息
3. `job_search.log` - 包含程序运行日志

## 常见问题

1. **程序无法启动**
   - 确保您已安装Chrome浏览器
   - 确保您已安装Python 3.11或更高版本（如果使用方式二或三）
   - 确保所有依赖项已正确安装

2. **程序运行缓慢**
   - 这是正常现象，程序需要模拟人类操作，并等待网页加载
   - 确保您的网络连接稳定
   - 关闭不必要的应用程序以释放系统资源

3. **程序频繁重启**
   - 这是设计的一部分，程序会在遇到问题时自动重启
   - 查看`job_search.log`文件了解详细错误信息

4. **如何停止程序**
   - 关闭程序窗口或按Ctrl+C
   - 关闭Chrome浏览器窗口

## 注意事项

- 程序运行期间请勿手动操作浏览器
- 首次运行时可能需要登录您的LinkedIn账户
- 程序会记录已申请的工作，避免重复申请
- 如需自定义更多参数，请参考配置文件中的说明
- 程序可能需要运行较长时间才能完成100个申请
